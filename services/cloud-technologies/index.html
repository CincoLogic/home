AWS

Amazon Web Services (AWS) is a comprehensive and widely adopted cloud platform offered by Amazon. It provides a broad set of cloud-based services that cater to various needs, including computing power, storage, and databases, among others.

Google Cloud

Google Cloud Platform is a suite of cloud computing services offered by Google that provides a series of modular cloud services including computing, data storage, data analytics, and machine learning, alongside a set of management tools.

MicrosoftAzure

Microsoft Azure, commonly referred to as Azure, is a comprehensive cloud computing platform provided by Microsoft. It offers a wide range of cloud services including those for computing, analytics, storage, and networking.
